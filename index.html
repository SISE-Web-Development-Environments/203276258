Three differences between Web 1.0, Web 2.0 and Web 3.0:

Web 1.0
Web 1.0 just represents the time of the internet before web 2.0 became reality, and is mainly static general web sites, etc. Web 1.0 was consider as read-only web with very little interaction where consumer can exchange the information together but it was not possible to interact with the website.
Web 2.0
Web 2.0 is currently the "standard" web tech dominating web applications - basically it refers to highly interactive websites where you can post data and view it. It refers to read-write websites and the time of emergence of social networking websites like flicker, face book, YouTube etc. Ajax, Rss etc are the concepts derived from web2.0.Web 2.0 is a web as a platform where users can leave many of the controls they have used in web 2.0. In other words, the user of web2.0 has more interaction with less control. Web 2.0 is not only a new version of web 1.0; Flexible web design, creative reuse, updates, collaborative content creation and modification were facilitated through web 2.0 one of the outstanding feature of the web 2.0 is to support collaboration and to help gather collective intelligence rather web 1.0
Web 3.0
Web 3.0 is the next gen of typical web applications and evolution of Symantec web. It refers the evolution of web utilization and interaction which includes altering the Web into a database. In enables the upgradation of back-end of the web, after a long time of focus on the front-end (Web 2.0 has mainly been about AJAX, tagging, and another front-end user-experience innovation). Web 3.0 is a term which is used to describe many evolutions of web usage and interaction among several paths. In this, data isn’t owned but instead shared, where services show different views for the same web / the same data.

Use case- the query: "Corona virus symptoms"
-Web 1.0 return static pages with information about Corona virus symptoms.
-Web 2.0 return an interactive results such as Wikipedia pages, YouTube videos and facebook pages about Corona virus symptoms.
-Web 3.0 return dynamic results which based on the wisdom of the crowd. Those results are interactive pages that contain data which based on more than trillion user posts about Corona virus symptoms and in addition it's contain data that in context to any semantic change of this query.
